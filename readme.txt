== Installation ==

Installation should take less than 5 minutes.

Basic installation steps:

1. Upload the "CartStack" plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce > CartStack to configure basic settings
4. Get your unique Site ID in your CartStack account (https://admin.cartstack.com/setup/)

For more detailed instructions (with screenshots) visit our help article - http://help.cartstack.com/article/84-woocommerce.
